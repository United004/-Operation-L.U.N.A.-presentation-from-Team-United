# -Operation-L.U.N.A.-presentation-from-Team-United
Разведка Луны относится к задачам космических программ по изучению и картографированию поверхности Луны для будущих миссий, включая поиск воды и ресурсов. Это открывает новые перспективы для устойчивого присутствия человечества за пределами Земли.
